CRUD веб-приложение для хранения пользователей. Реализована авторизация с правами user и admin.
Реализована возможность хранения юзеров либо в БД postgresql, либо в хип ConcurrentHashMap.
Выбор хранения устанавливается в logic/ValidateService.
На странице addUser.jsp реализована динамическая загрузка списка стран и при выборе страны спска
городов. Для реализации используются технологии javascript, Jquery, ajax, json.
